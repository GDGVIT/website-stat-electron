# Website Stats

**A Desktop app to see all the details about your website.**

Libraries used
--------------
- Electron
- React
- Babel for ES6
- Browserify

How to Run?
-----------
`npm install` and `npm start`

To reflect your changes:
`npm run watch` will run browserify and write all changes to bundle.js
